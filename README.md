
## 1. О проекте

**VPT Learn** — это кроссплатформенное мобильное приложение для обучения программированию, разработанное в рамках дипломного проекта. Платформа предоставляет структурированные курсы, интерактивные уроки, практические задания (тесты и написание кода), отслеживание прогресса и ролевую модель (студент, преподаватель, администратор).

## 2. Цели и Задачи

**Цель:** разработать мобильное приложение для хранения и представления данных о языках программирования, учебных материалах и практических заданиях, необходимых для их изучения.

**Задачи:**
- Анализ предметной области и существующих аналогов (SoloLearn, Mimo, Programming Hub, Enki, Grasshopper, Codecademy Go)
- Проектирование архитектуры клиент-серверного приложения
- Реализация кроссплатформенного мобильного клиента на Flutter
- Реализация серверной части на ASP.NET Core с REST API
- Интеграция с базой данных Supabase (PostgreSQL)
- Разработка системы аутентификации на JWT-токенах
- Реализация ролевой модели (студент, преподаватель, администратор)
- Разработка системы выполнения кода в изолированной среде (Docker-песочница)
- Тестирование и оценка качества

---

## 3. Функциональность
**Для студентов:**
- регистрация и авторизация в системе
- просмотр каталога доступных курсов
- изучение теоретических уроков
- выполнение тестовых заданий с выбором ответа
- выполнение практических заданий с написанием кода
- запуск и проверка кода в изолированной среде
- просмотр справочной информации в базе знаний

**Для администраторов:**
- просмотр списка всех пользователей
- изменение ролей пользователей
- редактирование данных учётных записей
- полный доступ ко всем функциям системы

**Ограничения:**
- требуется стабильное интернет-соединение для работы с сервером
- кодовые задания проверяются только на сервере (нет локальной песочницы)
- ограниченная поддержка офлайн-режима
- тестовый режим эксплуатации (не для промышленного использования)

## 4. Архитектура проекта

Тип архитектуры: клиент-сервер

**Технологии:**
- Flutter (Dart) — мобильный клиент
- ASP.NET Core 8.0 (C#) — серверная часть
- Supabase (PostgreSQL) — база данных
- Docker — изолированная среда для выполнения кода
- JWT — аутентификация

**Структура проекта:**
- lib/ — клиентская часть (Flutter)
- Backend/ — серверная часть (ASP.NET Core)
- android/, ios/, windows/ — платформенные конфигурации
- docs/ — документация и скриншоты
- tests/ — тесты

## 5. Установка и запуск

**Требования:**
- Flutter 3.35.3+
- .NET SDK 8.0+
- Docker 24+ (для песочницы)
- Android Studio / VS Code (для эмуляции)

**Установка:**
```bash
git clone https://github.com/vpt-student-projects/dev-skills-trainer-app.git
cd dev-skills-trainer-app
flutter pub get
```

**Запуск:**
```bash
flutter run
```
